# Learning Progress Tracker

A collaborative learning progress tracker for Chayan and Divyam to manage and track learning topics.

## Features

- **Common Topics Pool**: Add topics that can be learned by either person
- **Individual Progress Sections**: Track topics assigned to Chayan and Divyam separately
- **Subtopic Tracking**: Break down topics into smaller learning units
- **Progress Indicators**: Visual progress bars and completion percentages
- **Dark Mode Support**: Toggle between light and dark themes
- **Interactive UI**: Smooth animations and responsive design

## Tech Stack

- **Frontend**: React 18 with TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: Express.js with Node.js
- **Database**: PostgreSQL with Drizzle ORM
- **API**: RESTful API with React Query for state management

## Getting Started

```bash
npm install
npm run dev
```

The application will be available at `http://localhost:5000`

## Project Structure

- `client/src/` - Frontend React application
- `server/` - Express backend and API routes
- `shared/` - Shared types and schemas
- `db/` - Database configuration
